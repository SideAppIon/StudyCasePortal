Правила безопасности Firebase

Для загрузки картинок и логотипа нужно включить Firebase Storage.
Перейди в Firebase → Storage → Get started → выбери регион → Rules → вставь:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      allow read: if request.auth != null;
      allow write: if request.auth != null;
    }
  }
}
